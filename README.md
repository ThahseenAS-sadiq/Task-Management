# Task Management 🗂️

A modern **Task Management Web Application** that allows users to create, search, update, and delete tasks easily.
This project demonstrates **CRUD operations** with a clean and responsive user interface.

---

# 🚀 Features

* Create new tasks
* Edit existing tasks
* Delete tasks
* Search tasks quickly
* Task status management
* Clean and responsive UI

---

# 🖥️ Live Demo

Frontend deployed on Vercel.

# 💻 Installation

Clone the repository

```
git clone https://github.com/ThahseenAS-sadiq/task-Management.git
```

Move into project folder

```
cd task-Management
```

Install dependencies

```
npm install
```

Run the project

```
npm run dev
```
